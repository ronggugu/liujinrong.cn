Rust 基础入门

安装 Rust

首先，我们需要安装 Rust。可以从官方网站 https://www.rust-lang.org/tools/install 下载并安装 Rust。

Windows

下载安装包，安装即可。

Linux && Mac

安装完成后，可以使用 rustc 命令行工具来编译 Rust 程序。此外，Rust 社区还提供了许多 IDE 和编辑器插件，如 Visual Studio Code 的 Rust 插件，可以大大提高开发效率。
